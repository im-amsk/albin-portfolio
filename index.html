Featured Projects

Some projects and ideas I've worked on recently.

Combat-Based Turn Game

A Java command-line turn-based combat game designed using Object-Oriented Design Principles and SOLID architecture. The project follows the Boundary-Control-Entity (BCE) architectural pattern with strong separation of concerns.

  • Dynamic action filtering based on player state
  • Buff/Debuff categorisation system
  • Centralised input validation with recovery handling
  • Mid-combat restart and exit gateway system
  • Built entirely in Java

SepsiPatch & SepsiCare Plus

A healthcare-focused monitoring solution for sepsis survivors, designed to continuously monitor patients and detect possible sepsis recurrence while enabling them to live more freely and safely.

  • Agentic health alerts and smart suggestions
  • Biodegradable micro-needle monitoring layer
  • Biomarker and PTT-based monitoring concepts
  • Shortlisted in the OpenClaw Agentic Ideathon
